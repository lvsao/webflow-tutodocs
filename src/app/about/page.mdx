# About This Project

Welcome to our Nextra documentation site deployed on Webflow Cloud!

## What is This?

This is a modern documentation site that combines the power of:

- **[Nextra 4.0](https://nextra.site/)**: A Next.js-based documentation framework
- **[Next.js 15](https://nextjs.org/)**: The React framework for production
- **[Webflow Cloud](https://webflow.com/cloud)**: Edge-native deployment platform
- **[Cloudflare Workers](https://workers.cloudflare.com/)**: Global edge computing

## Why This Stack?

### Developer Experience

Building documentation should be easy and enjoyable. With Nextra, you write content in MDX (Markdown + React), which means:

- **Simple Syntax**: Write in Markdown for basic content
- **React Components**: Add interactive elements when needed
- **Type Safety**: Full TypeScript support
- **Hot Reload**: See changes instantly
