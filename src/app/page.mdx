## Quick Start

Get your documentation site running in 3 simple steps:

```bash
# 1. Install dependencies
npm install

# 2. Run development server
npm run dev

# 3. Deploy to Webflow Cloud
npm run deploy
```

<Callout type="warning">
  Make sure you have Node.js 18+ and a Webflow Cloud account before starting.
</Callout>
