# Getting Started

This guide will help you get started with this documentation site.

## Installation

The site is built with the following technologies:

- **Next.js 15**: The React framework
- **Nextra 4**: Documentation framework
- **Webflow Cloud**: Deployment platform
- **Cloudflare Workers**: Edge computing

## Configuration

The site is configured to work with Webflow Cloud's requirements:

```typescript
// next.config.ts
const nextConfig: NextConfig = {
  basePath: "/docs",
  assetPrefix: "/docs",
};
```

## Development

To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000/docs](http://localhost:3000/docs) in your browser.

## Building

To build for production:

```bash
npm run build
```

## Deployment

Deploy to Webflow Cloud:

```bash
npm run deploy
```

That's it! Your documentation site is now live on Webflow Cloud.
